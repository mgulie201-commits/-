В программе создаётся основной Класс Animal. В программе - описываеться общее поведение и свойства животных. У каждого животного есть имя (name), возраст (age) и вес (weight). Эти данные задаются при создании объекта через конструктор __init__. Также внутри класса есть скрытое поле __health, которое показывает состояние здоровья животного и считается приватным (его нельзя напрямую использовать вне класса).
Класс Animal содержит несколько методов.
Метод speak() отвечает за звук животного, но в базовом классе он возвращает "...", потому что у разных животных звук разный.
Метод eat(food) показывает, что животное ест определённую еду.
Метод info() возвращает строку с информацией о животном (имя, возраст и вес).
Метод __str__() позволяет красиво выводить информацию об объекте при использовании функции print().
У каждого класса есть метод: класс Cat метод speak() - возвращает "Мяу!"
дополнительный метод purr() показывает, что кошка мурлычет.
Class dog метод speak() - возвращал "Гав!"
дополнительный метод fetch(), - собака приносит палку.
Class Parrot метод speak() возвращает "Привет!"
метод repeat(phrase) - позволяет повторять любую переданную фразу.
Таким образом, программа показывает передача методов и создание разных типов обьектов животных
которые имеют общие свойства, но ведут себя по-разному.
